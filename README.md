# CluckTalk

CluckTalk is an AI-enabled cross-platform mobile application for poultry vocalization education, welfare monitoring, and precision livestock farming.

Developed by the MooAnalytica Research Group at Dalhousie University, CluckTalk enables users to explore annotated chicken vocalizations and learn their behavioural and welfare significance through an interactive mobile interface.

## Features

- Chicken vocalization library
- Welfare-oriented call interpretation
- AI-assisted poultry bioacoustics
- Educational learning platform
- Precision poultry monitoring

## Research Applications

- Poultry welfare assessment
- Precision livestock farming
- Animal behaviour research
- Bioacoustics and machine learning
- Digital agriculture
